🔐 Password Strength Checker:
  A modern, real-time password strength checker that evaluates security using entropy-based analysis, providing instant feedback, crack-time estimation, and actionable suggestions.

✨ Features:
  🔍 Real-time password strength analysis
  📊 Visual strength indicator (progress bar + color feedback)
  🧠 Entropy-based evaluation (not just rule-based)
  ⏱ Estimated time to crack
  🚫 Detection of common/weak passwords
  💡 Actionable suggestions to improve password strength
  🎨 Clean, modern, responsive UI
  🌙 Dark mode support

How It Works:
  The application evaluates password strength using:
    Length & character diversity
    Entropy calculation (log-based estimation of possible combinations)
    Pattern detection (repeated characters, sequences)
    Common password database matching
